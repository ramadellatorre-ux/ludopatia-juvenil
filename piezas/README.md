# Piezas de la campaña

La galería 3D del sitio busca acá los archivos:

    pieza-01.png … pieza-11.png

Mientras no estén, el sitio muestra un placeholder con el nombre de cada pieza
(funciona igual). Para usar las imágenes reales, exportá las 11 piezas desde el
proyecto de Claude Design "Campaña de ludopatía juvenil" (carpeta `piezas/`) y
dejalas acá con esos mismos nombres. Después: git add . && git commit && git push
